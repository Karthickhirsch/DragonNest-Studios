### Puzzle Types Summary
| Type | Example |
|------|---------|
| Symbol Match | Rotate tiles to match wall carving |
| Light Beam | Direct light to receiver using mirrors |
| Push & Weight | Shove blocks onto pressure plates |
| Sequence / Order | Activate switches in correct order |
| Environmental Observation | Decode shadows, constellations, carvings |
| Circuit / Flow | Connect pipe/wire segments |
| Elemental Interaction | Use abilities (Fire/Ice/etc.) on objects |
| Timing & Rhythm | Act in sync with moving hazards |
| Sound-Based | Match tones, follow echoes |
| Memory Sequence | Watch and reproduce a sequence |

## Engine & Tech Stack
- **Engine:** Unity 2022 LTS or Unity 6
- **Language:** C#
- **Render Pipeline:** URP (Universal Render Pipeline)
- **Audio:** FMOD
- **Camera:** Cinemachine
- **Input:** Unity New Input System
- **Cutscenes:** Unity Timeline
