# Model Specs — Island Environment Props
## Isle of Trials — Islands/

---

## How to Use These Specs

Each island needs a set of modular environment props that are placed in the scene to build the world.
You do NOT need to model the entire island as one piece. Instead build it like LEGO blocks —
place many small modular pieces together.

## Unity Import Checklist (Environment Props)
- [ ] All props in `Assets/_Game/Art/Models/Islands/IslandName/`
- [ ] Add MeshCollider or BoxCollider (where player will walk on/around it)
- [ ] Mark static objects as **Static** in Inspector (top right checkbox) for occlusion culling
- [ ] Set objects that player shouldn't walk through to Layer "Default"
- [ ] Create Prefab for each prop — drag from Scene to Project window
- [ ] Group variants: Rock_Small, Rock_Medium, Rock_Large (reuse same mesh, different scales)
