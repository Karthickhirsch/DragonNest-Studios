# Model Specs — All Enemies
## Isle of Trials — Enemies/

---

## Shared Style for All Enemies
All enemies use: **Low poly stylized 3D, Zelda Wind Waker aesthetic, vibrant island-themed colors, T-pose or neutral pose, white background, game-ready mesh**

---

## Enemy 1 — Driftwood Crab
**Island:** Tutorial (Driftwood Cay) | **Tier:** T1 | **Size:** ~0.6 x 0.6 x 0.4 units

### Visual Description
- Small crab made entirely of driftwood pieces lashed together with seaweed
- Asymmetric claws — one larger than the other
- Glowing amber eyes (2 small gems in wooden head)
- Mossy patches on shell
- Colors: Light tan/driftwood, green moss, amber eyes

### AI Prompt
```
Low poly stylized 3D game enemy, small crab creature made of driftwood and seaweed,
asymmetric claws one larger than the other, glowing amber gem eyes, mossy patches on body,
lashed-together wooden pieces for shell, 8 segmented wooden legs, animated crouching pose,
low poly Wind Waker style, light tan and brown driftwood colors with green moss accents,
white background, centered, game-ready mesh, 1000-2000 triangles, 1024x1024 texture
```

### Animations Needed
- Idle (claw snap), Walk (scuttle sideways), Attack (claw swipe), Death (collapse into wood pieces)

## Animation Download — Mixamo (For Humanoid Enemies)
For Jungle Ape and Storm Imp (humanoid rigs):
1. Go to https://www.mixamo.com
2. Upload your model FBX
3. Click "Auto-Rig"
4. Browse animations → select ones from the list above
5. Download as FBX with animation baked in

## Unity Import Checklist (All Enemies)
- [ ] FBX file in `Assets/_Game/Art/Models/Enemies/`
- [ ] Set Layer to **"Enemy"** on all enemy prefabs
- [ ] Add NavMeshAgent component
- [ ] Add Animator + attach AnimatorController
- [ ] Add Collider (CapsuleCollider for bipeds, BoxCollider for others)
- [ ] Add correct enemy script (EmberLizard.cs etc.)
- [ ] Assign EnemyData ScriptableObject in Inspector
- [ ] Scale to correct in-game size
